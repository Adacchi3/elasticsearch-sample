# Unit: 全文検索機能実装

## 概要
キーワードで記事を全文検索し、タイトル・本文抜粋・投稿日時を返す検索機能を Rails で実装する。

## 含まれるユーザーストーリー
- ストーリー 4: 全文検索

## 責務
- 検索フォームと結果表示ページの実装（`GET /articles/search?q=キーワード`）
- ElasticSearch の `multi_match` クエリでタイトル・本文を横断検索
- 検索結果の HTML ビュー表示（タイトル・本文抜粋・投稿日時、Tailwind CSS でスタイリング）
- 本文抜粋の生成（ElasticSearch highlight または アプリ側トリミング）
- キーワード空の場合・マッチなしの場合のハンドリング

## 境界
- ElasticSearch インデックスの設定・更新は Unit 003 が担当
- 記事の投稿・編集・削除 UI は含まない

## 依存関係

### 依存する Unit
- Unit 003: ElasticSearch インデックス登録（インデックスが存在すること）

### 外部依存
- `elasticsearch-model` gem
- ElasticSearch 8.x

## 非機能要件（NFR）
- **パフォーマンス**: 特になし（サンプル規模）
- **セキュリティ**: キーワードのサニタイズを行う
- **スケーラビリティ**: 特になし
- **可用性**: 特になし

## 技術的考慮事項
- レスポンスは JSON 形式で返す
- `multi_match` クエリで `title` と `body` を検索対象とする
- 本文抜粋は最大200文字程度にトリミングする
